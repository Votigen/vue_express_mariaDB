1. **SCM Project**

   **Introduction**

   A personal full-stack project using Node.js + Vue + UniApp.

   **Software Architecture**

   - Backend: Express + Node.js
   - Mobile App: UniApp + Uni UI
   - PC Frontend: Vue + Element UI

   **Installation Tutorial**

   1. Navigate to the `it_project` directory and run `cnpm i` (Ensure Node.js is installed beforehand).
   2. Navigate to the `server` directory and run `cnpm i`.
   3. Open the `scmproject` folder using Hbuilder.
   4. Import the provided SQL file into your MySQL database and name the database `scmproject`.

   **Usage Instructions**

   1. Ensure the backend server is running before starting the frontend.
   2. Running both the frontend and backend simultaneously might cause port conflicts.
   3. Please note that some features are not yet fully implemented.
